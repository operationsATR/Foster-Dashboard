Admin quick guide

Everything you need to manage the foster dashboard.

Adding a foster
Go to Manage tab
Enter their email + name
Fill in placement details — animal name, type, sex, birth date, foster start, Drive link, care instructions
Click Add another placement for multiple dogs
Save foster + placements — they can log in immediately
Adding key dates
Go to Foster profiles → select foster → scroll to Key dates
Click Add date → choose type, date, time, notes → Save
Appears on the foster's Litter tab immediately
Same type replaces the old date — no duplicates
Messaging fosters
Direct message — goes only to that foster, requires read confirmation
Board post — visible to all fosters in their Messages tab
Foster replies show under Sent direct messages with read status
Report cards
Auto-emailed to behavior@ and fostervolunteer@ when submitted — Word doc attached
Open the .docx in Google Docs to edit, then download as PDF to share
View all RCs in the foster's profile → mark reviewed when checked
Daily digest email
Sends to fostervolunteer@ each morning with updates, new RCs, upcoming dates, and quiet fosters
To activate: Apps Script → Triggers → Add Trigger → sendDailyDigest → Time-driven → Day timer → 8–9am
